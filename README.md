# browser-ai-agent
Очень простой консольный ИИ-агент для работы в браузере Goodle Chrome, написанный на Python и использующий локальную модель Ollama.
# Особенности
На данный момент работает в терминале, но планируется добавить полноценный веб-интерфейс
В качестве ИИ-модели используется локально запущенная llama 3.2:3b
Также планируется доработка для выполнения более сложных задач.
# Установка
1. Клонируйте репозиторий
git clone https://github.com/Hesperidium777/browser-ai-agent.git
2. Перейдите в папку с программой
cd browser-ai-agent
3. Установите зависимости
pip install -r requirements.txt
4. Запустите ollama, предварительно скачайте модель с официального сайта - https://ollama.com/library/llama3.2:3b
ollama pull llama3.2:3b
5. Проверьте работает ли модель
ollama serve
6. Запустите программу
python main.py
# Тестирование
Впишите в терминал какую-нибудь задачу, например: погода в Краснодаре
